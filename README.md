🚀 Многопоточный комбайн рассылок через Gmail

Скрипт на Python для массовой автоматизированной рассылки персонализированных писем. Проект разработан с упором на многопоточность, отказоустойчивость, валидацию входящих данных и информативное логирование.

🔥 Ключевые фичи и архитектура:

Параллельная отправка (threading): Письма отправляются в несколько потоков одновременно, что сводит время рассылки к минимуму.

Защита от блокировок (Semaphore): Количество одновременно активных потоков строго ограничено (не более 3), чтобы не превышать лимиты SMTP-сервера Gmail и избегать спам-фильтров.

Потокобезопасное логирование (Lock): Результаты записываются в файл report.txt. Использование механизма блокировок гарантирует, что потоки не затрут данные друг друга при одновременной записи.

Импорт базы из Excel (openpyxl): Скрипт автоматически парсит имена и email-адреса из внешней таблицы, исключая необходимость хардкодить контакты.

Строгая валидация (re): Регулярные выражения проверяют корректность каждого email-адреса. «Кривые» или пустые строки пропускаются автоматически, не вызывая падения всей программы.

Безопасность (python-dotenv): Конфиденциальные данные (логин и пароль приложения) вынесены в изолированные переменные окружения.

🛠️ Инструкция по установке и запуску:

Подготовка окружения
Склонируйте репозиторий (или скачайте файлы) и установите необходимые зависимости через терминал:
pip install openpyxl python-dotenv

Настройка конфигурации (.env)
Переименуйте файл-шаблон .env.example в .env и укажите свои данные авторизации:
EMAIL_USER=your_email@gmail.com
EMAIL_PASSWORD=your_16_digit_app_password

EMAIL_USER — Ваш реальный почтовый ящик Gmail, с которого пойдет рассылка.

EMAIL_PASSWORD — Специальный 16-значный пароль приложения, сгенерированный в аккаунте Google.

Подготовка базы контактов
Создайте файл contacts.xlsx в корневом каталоге проекта со следующей структурой:

Колонка A (строка 1): Name (Имя получателя)

Колонка B (строка 1): Email (Электронный адрес)
(Строки со 2-й и далее заполняются вашими данными).

Запуск программы
Запустите главный скрипт:
python smart-threaded-mailer-ru.py
